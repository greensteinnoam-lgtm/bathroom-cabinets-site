גרינשטיין ארונות אמבטיה - אתר סטטי

קבצים מרכזיים:
1. index.html - האתר הראשי, SEO, תוכן שיווקי, FAQ וטופס וואטסאפ.
2. products.html - דף נחיתת קטלוג קצר לפרסום מוצרים בלבד.
3. app.js - רשימת המוצרים, מחירים, צבעים, מידות וקישורי וואטסאפ.
4. styles.css - עיצוב האתר והקטלוג.
5. assets - תמונות מקומיות.

עריכת מוצרים:
פותחים את app.js ועורכים את המערך products.
לכל מוצר אפשר לשנות:
title, category, size, finish, price, priceBySize, description, sizes, colors, image.

שינוי מספר וואטסאפ:
בקובץ app.js מחפשים:
whatsappNumber: "972584209429"
ומחליפים למספר הרצוי בפורמט בינלאומי ללא + וללא 0 בהתחלה.

העלאה ל-Netlify:
1. מכווצים את הקבצים או משתמשים בקובץ bathroom-cabinets-site-live.zip.
2. נכנסים אל https://app.netlify.com/drop
3. גוררים את קובץ ה-ZIP.
4. אחרי החיבור לדומיין אמיתי, מומלץ לעדכן canonical ו-og:url בקבצים index.html ו-products.html.
